# **App Name**: FinSolve AI

## Core Features:

- Document Upload: Intuitive document upload via drag & drop or browse.
- Natural Language Input: Natural language query input via text or voice, with multilingual support.
- AI Answer Generation: AI-powered answer generation, using an LLM tool that will provide decision and quote document clauses used.
- Structured Results: Structured, card-based result output with decision, explanation, clause quote, and multilingual switch.
- Report History: My Reports dashboard to track previous documents and decisions.
- Help & Support: Help and feedback section with chatbot and FAQ.
- Navigation Bar: Sticky top navbar with logo, menu, and profile dropdown for easy navigation.

## Style Guidelines:

- Primary color: Deep blue (#0A3D91) for trust and reliability.
- Background color: Light blue (#E0F2F7) for a clean and airy feel.
- Accent color: Vibrant orange (#FFA500) for CTAs and important highlights.
- Headline font: 'Poppins' (sans-serif) for headers to create a modern and readable aesthetic.
- Body font: 'Inter' (sans-serif) for body text, offering clarity and legibility in a wide array of contexts.
- Use modern icons from Heroicons or Lucide for a consistent and clean user experience.
- Incorporate subtle animations for a smooth and engaging user experience.
